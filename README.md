```bash
# Clone the repository
$ git clone

# Access the project folder at the command prompt
$ cd ecommerce-store

# Install the dependencies
$ npm install

# Run the script "start"
$ npm run dev

# The project will start at the door: 5173 - access http://localhost:3000
```
